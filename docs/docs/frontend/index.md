# Frontend Application

The web application is built with **Next.js 16** and serves as the primary interface for our "Uber-like" moving platform.

## Ongoing Features

### 1. Blog Platform

A fully functional blog for sharing moving tips, success stories, and platform updates.

- **Stack**: Tiptap Editor for content creation.
- **Features**: Rich text editing, image embedding, server-side rendering for SEO.

### 2. Booking System

The core engine allowing users to schedule moves.

- **Status**: In development.
- **Goal**: Seamless step-by-step booking flow (Origin -> Destination -> Inventory -> Quote).

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS v4
- Clerk Auth
- Prisma (PostgreSQL)
- Tiptap Editor

## Key Commands

- `npm run dev`: Start dev server (Port 3000)
- `npm run build`: Build for production
